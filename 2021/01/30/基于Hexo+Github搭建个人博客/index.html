备注

若以下每一步操作没能实现期望效果，则需：

  1. hexo clean清除缓存文件和已生成的静态文件

  2. hexo g重新生成静态文件

  3. hexo s重新启动服务器

  4. 重新刷新浏览器即可

  5. 若还未达到期望值，则需百度。
